# Beethoven Op. 14 No. 2 검증 체크리스트

## Renderer 검증

- [ ] 대보표(Grand Staff) 렌더링이 올바르게 표시됨 (높은음자리표 + 낮은음자리표)
- [ ] 2/4 박자표가 첫 마디에 정확히 표시됨
- [ ] G장조 조표(#1개)가 올바르게 렌더링됨
- [ ] Allegro 템포 표시가 악보 상단에 표시됨
- [ ] 셈여림(p, f, sf 등) 기호가 올바른 위치에 표시됨
- [ ] 총 6페이지가 생성되고 페이지 넘김이 적절한 위치에서 발생함

## Restoration 검증

- [ ] 원본 이미지가 정상적으로 로드됨
- [ ] 그레이스케일 변환이 올바르게 수행됨
- [ ] 기울기 보정(deskew)이 적용됨
- [ ] 그림자 제거가 정상 작동함
- [ ] 대비 향상이 적용됨
- [ ] 이진화(binarization) 품질이 0.8 이상
- [ ] 전체 품질 점수가 0.7 이상
- [ ] 페이지 경계 감지가 정확함

## OMR 준비 검증

- [ ] MusicXML 파일이 정상적으로 파싱됨
- [ ] 음표 수가 예상 범위(800~1200) 내에 있음
- [ ] 쉼표 수가 예상 범위(50~150) 내에 있음
- [ ] 화음(chord)이 올바르게 인식됨
- [ ] 붙임줄(tie)과 이음줄(slur)이 구분되어 인식됨
